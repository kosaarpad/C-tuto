Összetett adatok: 2D tömbök (mátrixok)

Ha az 1D tömbökről szóló fejezetet nem olvastad még el, akkor előbb olvasd el azt, és csak utána térj ide vissza.
A 2D tömb (mátrix) alatt olyan tömböt értünk, amelyben az adatok egy adott sor és oszlopszámú táblázatba vannak rendezve. Minden adat a táblázatban ugyanolyan típusú kell legyen itt is, mint az 1D tömbök esetén. Az alábbiakban bemutatunk egy ilyen 2D tömbre egy példát. Ez egy a m azonosítójú tömb lesz, mely 3 sort és 4 oszlopot tartalmaz:

0. oszlop1. oszlop2. oszlop3. oszlop
0. sor m[0,0]m[0,1]m[0,2]m[0,3]
1. sor m[1,0]m[1,1]m[1,2]m[1,3]
2. sor m[2,0]m[2,1]m[2,2]m[2,3]

A sor- és oszlop sorszámozás (indexelés) 0-val kezdődik.
Minden elem tehát m[i, j] alakú, ahol az i a sorindex, j az oszlopindex.

	
using System;

class Program
{
	static void Main()
		{

		int [,] m = {
									{0, 1, 2, 3} , /* a 0. indexű sor értékei */
									{4, 5, 6, 7} , /* az 1. indexű sor értékei */
									{8, 9, 10, 11} /* a 2. indexű sor értékei */
								};
		}
}

2D tömb elemeinek kilistázása

Bemutatjuk egy példaprogramon keresztül:

	
using System;

class Program
{
	static void Main()
		{
		int [,] m = {
									{0, 1, 2, 3} , /* a 0. indexű sor értékei */
									{4, 5, 6, 7} , /* az 1. indexű sor értékei */
									{8, 9, 10, 11} /* a 2. indexű sor értékei */
								};
		int sor, oszlop;
		for(sor = 0;sor < 3; sor++)
				for(oszlop = 0;oszlop < 4; oszlop++)
					{
					Console.WriteLine("m["+sor+","+oszlop+"] = "+m[sor, oszlop]); 
					}
		}
}

Kimenet:

m[0,0] = 0
m[0,1] = 1
m[0,2] = 2
m[0,3] = 3
m[1,0] = 4
m[1,1] = 5
m[1,2] = 6
m[1,3] = 7
m[2,0] = 8
m[2,1] = 9
m[2,2] = 10
m[2,3] = 11

Adattábla szerint elrendezett kiírás:

	
using System;
class Program
{
	static void Main()
		{
		int [,] m = {
									{0, 1, 2, 3} , /* a 0. indexű sor értékei */
									{4, 5, 6, 7} , /* az 1. indexű sor értékei */
									{8, 9, 10, 11} /* a 2. indexű sor értékei */
								};
		int sor, oszlop;
		Console.WriteLine();
		for(sor = 0;sor < 3; sor++)
			{
			for(oszlop = 0;oszlop < 4; oszlop++)
				{
				Console.Write("m["+sor+","+oszlop+"] = "+m[sor, oszlop]+", "); 
				}
			Console.WriteLine();	
			}
		}
}

Kimenet:

m[0,0] = 0, m[0,1] = 1, m[0,2] = 2, m[0,3] = 3,
m[1,0] = 4, m[1,1] = 5, m[1,2] = 6, m[1,3] = 7,
m[2,0] = 8, m[2,1] = 9, m[2,2] = 10, m[2,3] = 11,

Feladat: készítsünk órarend kiíró programot!

	
using System;
class Program
{
	static void Main()
		{

		string [,] orarend = {
									{"sorsz", "Hétfő", "Kedd", "Szerda", "Csütörtök", "Péntek"} , /* a 0. indexű sor: a napok  */
									{"1.", "Angol","Matek" ,"Fizika" ,"Info", "Rajz" } , /* az 1.sorban az 1. órák */
									{"2.", "Matek" ,"Angol","Kémia" ,"Info", "Földrajz"}, /* a 2. sorban a 2. órák */
									{"3.", "Töri" ,"Fizika","Kémia" ,"Nyelvtan", "Tesi"}, /* ...stb*/
									{"4.", "Info" ,"Biológia","Német" ,"Irodalom", "Tesi"}, /* ...stb*/
									{"5.", "Info" ,"Kémia","Angol" ,"Irodalom", "Osztfőn"} /* ...stb*/
								};
		int sor, oszlop;
		Console.Clear();//törli a konzol tartalmát:
		for(sor = 0;sor < 6; sor++)
			{
			for(oszlop = 0;oszlop < 6; oszlop++)
				{
				Console.SetCursorPosition(5+oszlop*12, 3+sor*2);//adott oszlop adott sorába pozicionálja a kurzort
				Console.Write(orarend[sor, oszlop]+" "); 
				}
			}
		Console.ReadLine();	
		Console.WriteLine();	
		Console.WriteLine();	
		}
}

Kimenet:

   sorsz       Hétfő       Kedd        Szerda      Csütörtök   Péntek

    1.          Angol       Matek       Fizika      Info        Rajz

    2.          Matek       Angol       Kémia       Info        Földrajz

    3.          Töri        Fizika      Kémia       Nyelvtan    Tesi

    4.          Info        Biológia    Német       Irodalom    Tesi

    5.          Info        Kémia       Angol       Irodalom    Osztfőn
