Az if utasítás

Szerkezete: 

	
if (logikai_feltétel) 
	utasítás;

Magyarul kb így olvashatjuk: „Ha a feltétel igaz, akkor hajtsd végre az utasítást!”

Ha több utasítást kell végrehajtani, akkor az utasítás helyett utasításblokk szerepel: 
	
if (logikai_feltétel)
   {
    utasítás1;
    utasítás2;
    utasítás3;
    …
    }

Először a logikai feltétel értékelődik ki. Az utasításblokk tehát csak akkor kerül végrehajtásra, ha a logikai feltétel igaznak bizonyul.
Példa: 

	
using System;
class Program
{
    static void Main()
    {
	int x;
	x=100;
	if (x == 100) 
		Console.WriteLine("x értéke 100");	
    }
}

Itt ugye ellenőrzésre kerül, hogy x egyenlő.e 100-zal. Mivel x értéke tényleg 100, a feltétel igaz lesz, a program kiírja az "x értéke 100" üzenetet.
Nézzük meg, mit ad a fenti program, ha x értéke nem 100? 

	
using System;
class Program
{
    static void Main()
    {
	int x;
	x=101;
	if (x == 100) 
		Console.WriteLine("x értéke 100");	
    }
}

Ebben az esetben az if utáni utasítás nem hajtódik végre, mivel a 101 == 100 logikai feltétel hamis. Ha több utasítás áll az if után, akkor kapcsos zárójeleket alkalmazunk, utasításblokkba zárva az utasítások sorozatát: 

	
using System;
class Program
{
    static void Main()
    {
	int x;
	x=101;
	if (x == 100) 
	    {
		Console.WriteLine("x értéke");
		Console.WriteLine("100");	
		}
    }
}

Még egy példa:

	
using System;
class Program
{
    static void Main()
    {
	int jegy;
	Console.WriteLine("Hányas lett a programozás dogád?");
	jegy = Convert.ToInt32(Console.ReadLine());
	if (jegy > 4) 
		Console.WriteLine("jeles!");	
    }
}
