Adatok kiírása konzolra

Egyszerű adatok kiírása

A Console.Write(adat) vagy Console.WriteLine(adat) utasítással történik.
Itt az adat bármilyen egyszerű adat lehet, azaz int, double, bool, char, string. A Console.Write(adat) hatására az adat kiírása megtörténik, és a kurzor az utolsó karakter mögött marad.
Ha megint kiírásra kerül valami, akkor azt közvetlenül melléírja.
A Console.WriteLine(adat) hatására az adat kiírása megtörténik, és a kurzor a következő sorba ugrik. A következő adat már az új sor elejére kerül.
A Console.Write(adat) utasítást egymás mellé írásra,
a Console.WriteLine(adat) utasítást egymás alá írásra használjuk. 

using System;

public class Test
{
  static void Main()
		{
		int a = 34;
		double b = 2,23;
		char betu = 'B';
		boolean hidegVan = true;
		string sz = "alma";
		Console.WriteLine(a);//→ 34
		Console.WriteLine(b);//→ 2,23
		Console.WriteLine(betu);//→ 'B'
		Console.WriteLine(hidegVan);//→ true
		Console.WriteLine(sz);//→ "alma"
		}
}

Tömb kiírása konzolra

Legegyszerűbb foreach ciklussal:

using System;

public class Test
{
  static void Main()
		{
		string[] gyumolcs = {"alma","körte","szilva","barack","szőlő"};
		foreach (string gyumi in gyumolcs)
			{
			Console.WriteLine(gyumi);
			}			
		}
}

A fenti kiíratás működik listákra, halmazokra is.

using System;
using System.Collections.Generic;

public class Test
{
  static void Main()
		{
		List <string> gyumolcs = new List<string>();
		gyumolcs.Add("alma");
		gyumolcs.Add("korte");
		gyumolcs.Add("szilva");
		gyumolcs.Add("barack");
		//kiíratás:
		foreach (string gyumi in gyumolcs)
			{
			Console.WriteLine(gyumi);
			}			
		}
}

Egyszerű adat bekérése konzolról

A szoveg = Console.ReadLine() utasítással történik.
Ezzel az utasítással csak szöveges adatot tudunk bekérni.
Amennyiben számra vagy egyéb adatra van szükségünk, úgy adatkonverzióra lesz szükségünk.
Adatbekéréskor illik tájékoztatni a felhasználót, hogy mit is várunk el tőle.

Szöveges adat bekérése konzolról

using System;

public class Test
{
  static void Main()
		{
		string nev;
		//üzenünk a felhasználónak
		Console.WriteLine("kérem, írja be a nevét:");
		//beolvassuk a begépelt adatot nev változóba
		nev = Console.ReadLine();
		//kiírunk:
		Console.WriteLine("Az ön neve: " + nev);
		}
}

Egész szám bekérése konzolról

Mint említettem, a konzolról csak szöveges információt olvashatunk be. Ezért aztán szükség van a szöveg számmá konvertálására:

using System;

public class Test
{
  static void Main()
		{
		string szoveg;
		int eletkor;
		//üzenünk a felhasználónak
		Console.WriteLine("kérem, írja be az életkorát:");
		//beolvassuk a begépelt adatot  a szoveg változóba
		szoveg = Console.ReadLine();
		//konvertáljuk:
		eletkor = Convert.ToInt32(szoveg);
		//kiírunk:
		Console.WriteLine("Az ön neve: " + nev);
		}
}

Ez a művelet a programfutás szempontjából kockázatos lehet, mert a felhasználó olyan adatot is megadhat, ami nem értelmezhető egész számként. Ezért az ilyen műveleteket kivételkezeléssel ajánlott megcsinálni.
