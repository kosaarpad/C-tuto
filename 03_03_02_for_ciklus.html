C# középiskolásoknak

A For ciklus

Formátuma:

 
for (inicializálás; logikai_feltétel; ciklusváltozó_módosítás)
     ismételendő_utasítás;
vagy ha a ciklusmag több utasításból áll: 
 
for (inicializálás; logikai_feltétel; ciklusváltozó_módosítás)
     { 
      ismételendő_utasítás1;
      ismételendő_utasítás2;
      ismételendő_utasítás3;
	  ...
      ismételendő_utasításN;
     }

A fenti példákban a pirossal kiemelt rész a ciklusmag.
A for ciklus úgy ismételteti a ciklusmagot, hogy közben számlálja is az ismétléseket. A számláláshoz használt változót (vagy változókat) ciklusváltozóknak nevezzük.
Lássunk egy egyszerű példát: 

	
using System;
class Program
{
	static void Main()
	{
	int i;
	for (i=0; i < 10; i++)
		{
		Console.WriteLine(""+i);
		Console.WriteLine("Szia");
		}
	}
}

Lássuk tehát, hogyan működik ez a ciklus:

  1. Először a ciklusszámlálót (i-t) inicializáljuk, ami azt jelenti, hogy i megkapja a kezdőértékét, azaz a 0-t
  2. Utána kiértékelődik a logikai feltétel: i < 10 . Ha nem igaz, akkor a ciklus véget ér. Ha viszont igaz, akkor
  3. Végrehajtódik a ciklusmag
  4. A ciklusmag végrehajtása után a ciklusváltozót automatikusan módosul. Itt a módosuláss azt jelenti, hogy i-t növeljük 1-gyel: i++;
    Ezután a 3. ponthoz ugrunk.

Készíthetünk olyan ciklust is, amelyben a ciklusszámláló nem egyesével számlál:

	
using System;
class Program
{
	static void Main()
	{
	int i;
	for (i=0; i < 15; i+=3)
		Console.Write(""+i+",");
	}
}

Ez a program az i ciklusszámlálót írja ki minen ismétléskor. Az i-t 3-mal növeli, majd megint kiírja az i-t, egészen addig, amíg az i < 15 feltétel igaz marad.
A számlálós ciklussal a ciklusváltozót visszafele is számláltathatjuk: 

	
using System;
class Program
{
	static void Main()
	{
	int i;
	for (i=14; i >= 0; i--)
		Console.Write(""+i+",");
	}
}

Ez a program az is az i ciklusszámlálót írja ki minen ismétléskor. Csakhogy itt az i első értéke 14, majd 1-esével csökken 0-ig. Ha az i-t 3-asával akarjuk csökkenteni: 

	
using System;
class Program
{
	static void Main()
	{
	int i;
	for (i=14; i >= 0; i-=3)
		Console.Write(""+i+",");
	}
}

Egyszerre több ciklusváltozót is használhatunk:

	
using System;
class Program
{
	static void Main()
	{
	int i, n;
	for ( n=0, i=10 ; n!=i ; n++, i-- )
		 Console.WriteLine("n = "+n + " i = " + i);
	}
}

Kimenet:

n = 0 i = 10
n = 1 i = 9
n = 2 i = 8
n = 3 i = 7
n = 4 i = 6

Itt az n nő 0-ról egyesével, az i csökken 10-ről egyesével, és addig ismétlődik a kiírás, amíg n nem egyenlő i-vel.

A ciklusváltozó nem csak egész szám lehet...

...hanem pl. karakter vagy felsorolás is. Lássunk egy példát karakteres számlálós ciklussal: 

	
using System;
class Program
{
	static void Main()
	{
	char betu;
	for ( betu='A'; betu < 'G';betu++ )
		 Console.Write(betu+",");
	}
}

Ez a program kiírja egymás mellé az 'A, B, C, D, E, F' karaktersorozatot, egymástól vesszővel elválasztva.

Egymásba ágyazott for ciklusok

Sokszor kell valamilyen értéket egy másikkal párosítani.
példál betűkhöz értékeket. Tudjuk, hogy a következő órán valaki a csoportból felelni fog fizikából.
A csoport tagjai:
Aladár, Béla, Cili, Dóra, Elemér, Feri.
Írjunk programot, amely kiírja a felelési esemény összes lehetséges kimenetelét!
Felelhet ugye Aladár, és 1-est kaphat (A---1), de kaphat 2-est (A---2), 3-ast...stb.
vagy
Felelhet Béla és, 1-est kaphat (B---1), de kaphat 2-est (B---2), 3-ast...stb.
...stb.
A feladat tehát kiírni az összes lehetséges eseményt, vagyis (betű---jegy) kombinációt. 

	
using System;
class Program
	{
	static void Main()
		{
		char betu;
		int jegy;
		for ( betu='A'; betu < 'G';betu++ )
			{
			for ( jegy=1; jegy <= 5;jegy++ )
				Console.Write(betu+"---"+jegy+"  ");
			Console.WriteLine();//betűváltás után új sor
			}
		}
	}

Kimenet:

A---1  A---2  A---3  A---4  A---5
B---1  B---2  B---3  B---4  B---5
C---1  C---2  C---3  C---4  C---5
D---1  D---2  D---3  D---4  D---5
E---1  E---2  E---3  E---4  E---5
F---1  F---2  F---3  F---4  F---5
