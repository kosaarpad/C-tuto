C# középiskolásoknak

A konstruktor kulcsfontosságú fogalom az objektum-orientált programozásban.
  • A konstruktor az osztály egy speciális metódusa (tagfüggvénye), amelynek szerepe az osztályváltozók ellátása kezdőértékkel.
  • A neve megegyezik a az osztály nevével.
  • Automatikusan végrehajtódik, amikor példányosítjuk az osztályt, azaz amikor létrehozunk egy új objektumot.
  • A konstruktornak soha sincs visszatérési értéke (-> nincs return)
  • Ha nem hozunk létre mi kostruktort, a rendszer létrhoz automatikusan egy úgynevezett implicit konstruktort, amit default konstruktornak nevezünk

Példa konstruktorra

using System;
public class Téglalap
{
double hossz, szél;
//konstruktor
public Téglalap(double ujhossz, double ujszél)
	{
	this.hossz=ujhossz;
	this.szél=ujszél;
	}
}
class Program
{
// Main metódus
static void Main(string[] args)
	{
	Téglalap t1 = new Téglalap(4,1);//egy téglalap példány
	Téglalap t2 = new Téglalap(7,3);//egy másik téglalap példány
	Console.ReadKey();
	}
}

A copy konstruktor

A copy konstruktor egy olyan konstruktor, amely lemásolja egy másik objektum változóit. A C#-ban a copy konstruktornak is van paramétere, mégpedig egy ugyanolyan osztálytípusú paramétere, amilyent másolni akarunk. A copy konstruktor akkor hasznos, ha már egy létező osztályból akarunk egy újat létrehozni a már létező osztály változóinak értékeivel. A copy konstruktor átadja a mintaosztály változóértékeit az új osztálynak, anélkül, hogy a mintaosztályt módosítaná. Lássuk egy példán keresztül, hogyan is működik a copy konstruktor: 

using System;
public class Téglalap
{
double hossz, szél;
//sima konstruktor:
public Téglalap(double ujhossz, double ujszél)
	{
	this.hossz=ujhossz;
	this.szél=ujszél;
	}
}
//copy konstruktor:
public Téglalap(Téglalap minta)

{
this.hossz=minta.hossz;
this.szél=minta.szél;
}
class Program
{
// Main metódus
static void Main(string[] args)
	{
	Téglalap t1 = new Téglalap(4,1);//egy téglalap példány
	t2 = t1;//egy másik téglalap példány 
	//t2 ugyanolyan hosszúságú és szélességű, mint t1:
	Console.WriteLine(t2.hossz);
	Console.WriteLine(t2.szél);
	Console.ReadKey();
	}
}
