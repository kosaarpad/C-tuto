Egymásba ágyazott if ... else utasítások

Az if…else utasítások egymásba ágyazhatók, ha több, mint két irányba ágaztatjuk el a végrehajtást: Például:

	
using System;
class Program
{
static void Main()
	{
	int szam;
	Console.WriteLine("Kérek egy egész számot:");
	szam = Convert.ToInt32(Console.ReadLine());
	if (szam < 0)
		Console.WriteLine("A szám negatív!");	
	else if (szam ==0)
		Console.WriteLine("A szám nulla!");	
	else 
		Console.WriteLine("A szám pozitív!");	
	}
}

Az utolsó else ág nem tartalmaz feltételt, mert ha a szám nem negatív és nem nulla, akkor csak pozitív lehet.

	
using System;
class Program
{
	static void Main()
	{
	int jegy;
	Console.WriteLine("Hányas lett a programozás dogád?");
	jegy = Convert.ToInt32(Console.ReadLine());
	if (jegy==1)
		Console.WriteLine("Ez elégtelen!");	
	else if (jegy==2)
		Console.WriteLine("Ez elégséges!");	
	else if (jegy==3)
		Console.WriteLine("Ez közepes!");	
	else if (jegy==4)
		Console.WriteLine("Ez jó!");	
	else if (jegy==5)
		Console.WriteLine("Ez jeles!");	
	}
}

Egy tipikus eset, ahol egy kiszámított érték szerint többféleképpen értékelünk, a testtömegindex program: 

	
using System;
class Program
{
    static void Main()
    {
	double testmag, testsuly, tti;
	Console.WriteLine("Hány kiló vagy?");
	testsuly = Convert.ToDouble(Console.ReadLine());
	Console.WriteLine("Hány centi magas vagy?");
	testmag = Convert.ToDouble(Console.ReadLine());
	//átalakítjuk a testmag-ot cm-ből m-be:
	testmag = testmag/100.0;
	tti = testsuly/(testmag*testmag);
	if (tti < 15)
		Console.WriteLine("Anorexiás vagy!!! Fordulj orvoshoz!");
	else if (tti < 18) 
		Console.WriteLine("Kórosan sovány vagy!!!");
	else if (tti < 20) 
		Console.WriteLine("Sovány vagy!!!");
	else if (tti < 25) 
		Console.WriteLine("Átlagos a testalkatod!");
	else if (tti < 28) 
		Console.WriteLine("El vagy hízva!");
	else if (tti < 30) 
		Console.WriteLine("Súlyos elhízás!!! Fordulj orvoshoz!");
    }
}
