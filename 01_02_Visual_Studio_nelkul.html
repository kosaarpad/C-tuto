C# középiskolásoknak

Mi van, ha nincs Visual Studiónk?

Amennyiben nincs fejlesztőeszközünk, egy sima szövegszerkesztő eszköz is megteszi. Pl. a sima jegyzettömbbel vagy Notepad++-szal is megírhatjuk a programunkat.
Ekkor azonban kell írnunk egy parancsfájlt, ami fordításkor megkönnyíti a munkánkat.
Ennek a parancsfájlnak legyen a neve: ford.bat

echo off
SET PATH=%PATH%;C:\Windows\Microsoft.NET\Framework64\v4.0.30319
set arg1=%1
shift
csc %arg1%

Itt a SET PATH... kezdetű sorban a csc.exe fordítóprogram elérési útvonalát kell megadnunk. Az így elkészült parancsfájl használata:

ford programnév.cs < ENTER >

Ennek hatására a programunk - amennyiben helyesen van megírva - lefordul. Vagyis lesz egy programnév.exe fájlunk. Ha a programunk hiba nélkül lefordult, a futtatáshoz írjuk be a program nevét, majd az ENTER billentyűt nyomjuk le. A programunk elindul, és amennyiben nem történik futási hiba, a program lefut.
