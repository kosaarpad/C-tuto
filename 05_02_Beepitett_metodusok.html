C# középiskolásoknak

Beépített ("gyári") metódusok(függvények)

A C# rendszer nagyon sok, előre megírt programkódot tartalmaz.
Ezeket a kódok előre megírt osztályok, amelyek megkönnyítik a munkánkat.
Az osztályok természetesen egy sor tagfüggvényt, metódust tartalmaznak. A string típusnál sok szövegkezelő metódussal találkoztunk már.
A valós típusoknál már tárgyaltunk a Math osztály metódusairól. Emlékeztetőül felsorolunk néhány beépített metódust, amit gyakran használhatunk: 

//A Math osztály gyakoribb metódusai
Math.Min(a,b); // a két változó közül a kisebb értéket adja eredményül
Math.Max(a,b); // a két változó közül a nagyobb értéket adja eredményül
Math.Round(tortszám, tizedes); // matematikai szabály szerint kerekíti a változó értékét
Math.Abs(a); // az adott változó értékének abszolút értékét adja vissza
Math.Sqrt(a); // az adott változó négyzetgyökét adja vissza
Math.Pow(a,b); // az a számot a b-edik hatványra emeli, vagyis kiszámítja ab-t 
Math.Log(a); //az a szám természetes logaritmusa, azaz logea, ahol e = 2,7182818...
Math.Log(a, b);//az a szám b alapú logaritmusa, vagyis logba
Math.Sin(a); // az a szám szinuszát adja vissza 
Math.Cos(a); // az a szám koszinuszát adja vissza 
Math.Tan(a); // az a szám koszinuszát adja vissza
Egynémely feladatnál hasznosak lehetnek a halmazok, és a halmazműveletek.
Ezt támogatja a System.Collection.Generics modul HashSet osztálya, amely többek között az alábbi függvényeket tartalmazza: 
//halmaz objektum gyakoribb metódusai

HasSet.Add(halmazelem); //a halmazhoz hozzáad egy elemet
HashSet.Remove(halmazelem); //halmazból kivesz egy elemet
HashSet1.UnionWith(HashSet2); // két halmaz uniója
HashSet1.IntersectWith(HashSet2); // két halmaz metszete
..stb.
A halmazhoz hasonlóképpen a lista is gyakran használt adatszerkezet, ennek is megvannak a saját metódusai. 
//lista objektum gyakoribb metódusai
List.Add(elem); //hozzáad egy elemet a listához
List.Remove(elem); //töröl egy elemet a listához
List.Insert(n, elem); //listaelemet az n-edik helyre beszúrja


..stb.
A listákkal, halmazokkal, szótárakkal a későbbiekben foglalkozunk.
