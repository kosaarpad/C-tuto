A foreach ciklus

A foreach ciklus egy speciális ciklus, amely segítségével eg tömb elemeit könnyen végigjárhatjuk.
Formátuma: 

foreach (elemtípus elemváltozó in tömbnév)
	csináljvalamit(eleváltozó);

	
using System;

class Program
{
	static void Main()
		{
		string[] kedvencek = { "kuty", "macs", "kanári", "aranyhal" };
		// ... Végigjárjuk és kiíratjuk a tömbelemeket:
		foreach (string allat in kedvencek)
			{
			Console.WriteLine(allat);
			}
		}
}

Ezt lehet foreach utasítás nélkül is csinálni, de így talán egyszerűbb. Íme a megoldás foreach nélkül:

	
using System;
class Program
{
	static void Main()
		{
		string[] kedvencek = { "kuty", "macs", "kanári", "aranyhal" };
		// ... Végigjárjuk és kiíratjuk a tömbelemeket:
		int letszam = kedvencek.Length;
		for (int i=0; i < letszam; i++)
			{
			Console.WriteLine(kedvencek[i]);
			}
		}
}

Egész tömb bejárása

Példa egészeket tároló tömbbel:

	
using System;

class Program
{
	static void Main()
		{
		int[] jegyek = { 3, 5, 2, 4 };
		// ... Végigjárjuk és kiíratjuk a tömbelemeket:
		foreach (int jegy in jegyek)
			{
			Console.WriteLine(jegy);
			}
		}
}

Valós tömb bejárása

	
using System;

class Program
{
	static void Main()
		{
		double[] szamok = { 3.2, -50.2, 201.3, -42.57 };
		// ... Végigjárjuk és kiíratjuk a tömbelemeket:
		foreach (double sz in szamok)
			{
			Console.WriteLine(sz);
			}
		}
}
