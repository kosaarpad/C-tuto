A logikai adattípus

Az adattípus = (A lehetséges értékek halmaza) + (az értékhalmazon végezhető műveletek)

A logikai típus, vagyis a bool segítségével olyan változókat valósíthatunk meg, amelyeknek mindössze 2 értékük van: true és false.
Ennek az adattípusnak az értékkészlete tehát 2 értékből áll, és a logikai műveletek végeszhetők el vele:

(false, true) + (&&, ||, !, ^)

A bool műveletkészlete tehát:
  • a && vagyis logikai ÉS
  • a || vagyis logikai VAGY
  • a ! vagyis logikai NEM
  • a ^ vagyis logikai KIZÁRÓ VAGY

A logikai NEM művelet: !

Egy bemenete van, tehát egy ítéletre hat.

Igazságtáblázat:

Be Ki
A !A
false true
true false

Példaprogram 

using System;
class Program
	{
    static void Main()
		{
		bool A = false;
		Console.WriteLine("Ha az A=" + A + " akkor !A = "+!A);
		A = true;
		Console.WriteLine("Ha az A=" + A + " akkor !A = "+!A);
		}
	}

És

Két bemenete van, tehát két ítéletet kapcsol össze. Csak akkor ad igazat, ha mindkét bemenet igaz

Jelölés: A && B


Példa ÉS műveletre:

kétjegyű = (szám > 9) &&(szám < 100)

Látható, hogy ahhoz, hogy a kétjegyűség igaz legyen, egyszerre mindkét feltételnek teljesülnie kell (vagyis értéke true kell legyen)

Igazságtáblázat:

Be1 Be2 Ki
A B A && B
false false false
false true false
true false false
true true true

Példaprogram 

using System;
class Program
	{
    static void Main()
		{
		bool A = false, B=false, C;
		C = A && B;
		Console.WriteLine("Ha A=" + A + ", B = " + B + " akkor A && B = " + C);
		A = false; B = true;
		C = A && B;
		Console.WriteLine("Ha A=" + A + ", B = " + B + " akkor A && B = " + C);
		A = true; B = false;
		C = A && B;
		Console.WriteLine("Ha A=" + A + ", B = " + B + " akkor A && B = " + C);
		A = true; B = true;
		C = A && B;
		Console.WriteLine("Ha A=" + A + ", B = " + B + " akkor A && B = " + C);
		}
	}

VAGY

Két bemenete van, tehát két ítéletet kapcsol össze. Csak akkor ad igazat, ha legalább az egyik bemenet igaz

Jelölés: A || B

Példa VAGY műveletre:

nemnulla = (szám < 0) || (szám > 0)

Látható, hogy ahhoz, hogy a kétjegyűség igaz legyen, egyszerre mindkét feltételnek teljesülnie kell (vagyis értéke true kell legyen)

true true true

KIZÁRÓ VAGY

Két bemenete van, tehát két ítéletet kapcsol össze. Csak akkor ad igazat, ha a két bemenet nem egyforma

Jelölés: A ^ B

Igazságtáblázat:

Be1 Be2 Ki
A B A ^ B
false false false
false true true
true false true
true true false

Példaprogram

	{
    static void Main()

		A = true; B = false;
		C = A ^ B;
		Console.WriteLine("Ha A=" + A + ", B = " + B + " akkor A ^ B = " + C);
		A = true; B = true;
		C = A ^ B;
		Console.WriteLine("Ha A=" + A + ", B = " + B + " akkor A ^ B = " + C);
		}
	}
