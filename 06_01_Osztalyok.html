Osztályok

Az osztály egy olyan adattípus, amely a dolgok tulajdonságait leíró adatok mellett tartalmazza a viselkedését is, amelyeket függvények segítségével adunk meg

A dolgokat jellemző mennyiségeket leíró adattagokat tulajdonságoknak nevezzük

A dolgok viselkedését leíró függvényeket (adattagokkal végezhető műveletek, a dolgok közötti kölcsönhatások) metódusoknak nevezzük

Az osztály lehetővé teszi azt, hogy a dolgok tulajdonságait és viselkedését egységes egészként kezeljük, ezáltal a programunk bonyolultságát csökkentsük.

Az osztályok használata ugyanakkor megvédi a programozót attól, hogy véletlenül egy bizonyos adatot felülírjon.

Osztály és objektum

  • Osztályok: az adatformátum és az elérhető metódusok definíciója az adott típus vagy a típushoz tartozó objektumok számára. Az osztályok is tartalmazhatnak adattagokat és metódusokat, amelyek műveleteket végeznek az osztály adattagjain. Összetartozó adatok és függvények, eljárások egysége.
  • Objektumok: az osztály példányai.

Osztály létrehozása

Két módunk van arra, hogy létrehozzunk egy osztályt. Az első: 
Osztálynév objektumnév = new Osztálynév();
Erre példa: 
Téglalap t = new Téglalap(2, 3);
A másik mód az, hogy egy már létező objektumból létrehozunk egy újat: 
Osztálynév objektumnév1 = new Osztálynév();
Osztálynév objektumnév2;
objektumnév2 = objektumnév1;
példa: 
Téglalap t1=new Téglalap(4,1);
Téglalap t2;
t2 = t1;
Ahhoz, hogy a fenti kódrészletek működjenek, természetesen szükség van arra, hogy megírjuk a Téglalap osztály definícióját.
Mi is egy téglalap? Milyen tulajdonságai vannak egy téglalapnak?
Előszöris, egy téglalapnak van hosszúsága meg szélessége.
Ezenkívül még lehet színe, kerülete, területe, stb.
Ha ismerjük a téglalapunk hosszúságát meg szélességét, a kerületét meg területét ki tudjuk számítani.
Ezért amikor létrehozzuk a téglalapot, elég ezt a két adatot megadni. 
using System;
public class Téglalap
{
double hossz, szél;
//konstruktor
public Téglalap(double ujhossz, double ujszél)
	{
	this.hossz=ujhossz;
	this.szél=ujszél;
	}
//tagfüggvény vagy metódus, ami a téglalap kerületét kiszámítja	
public double Kerület()
	{
	return 2*(this.hossz+this.szél);
	}
public double Terület()
	{
	return this.hossz*this.szél;
	}
public double Átló()
	{
	return Math.Sqrt(this.hossz*this.hossz+this.szél*this.szél);
	}	
}
class Program
{
// Main metódus
static void Main(string[] args)
	{
	Téglalap t1 = new Téglalap(4,1);
	Console.WriteLine("A téglalap kerülete: "+t1.Kerület());
	Console.WriteLine("A téglalap területe: "+t1.Terület());
	Console.WriteLine("A téglalap átlója: "+t1.Átló());
	Console.ReadKey();
	}
}
