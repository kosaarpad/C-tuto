Az if ... else utasítás

Szerkezete:

if (logikai_feltétel) 
	utasítás1; 
else 
	utasítás2;

Magyarul így olvassuk: ”Ha a logikai feltétel igaz, akkor hajtsd végre az utasítás1-et, ha nem, akkor hajtsd végre utasítás2-t!”
Példa: 

	
using System;
class Program
{
	static void Main()
	{
	int x;
	x=100;
	if (x == 100)
		Console.WriteLine("x erteke 100");
	else
		Console.WriteLine("x értéke NEM 100");	
	}
}

Ha több utasítás van az if vagy az else ágon, akkor utasításblokkba foglaljuk:

	
using System;
class Program
{
	static void Main()
	{
	int x;
	x=100;
	if (x == 100)
		{
		 Console.WriteLine("x erteke");
		 Console.WriteLine("100");
		}
	else
		{
		Console.WriteLine("x értéke"); 
		Console.WriteLine("NEM 100");	
		}
	}
}

Még egy példa:

	
using System;
class Program
{
    static void Main()
    {
	int jegy;
	Console.WriteLine("Hányas lett a programozás dogád?");
	jegy = Convert.ToInt32(Console.ReadLine());
	if (jegy > 4) 
		Console.WriteLine("jeles!");	
	else
		Console.WriteLine("ez nem jeles!");	
    }
}
