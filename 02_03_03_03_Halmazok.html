Halmazok

A System.Collections.Generic névtérben található egy HashSet osztály, amely az adatokat halmazszerűen tárolja.
A halmaz a listákhoz képest eltérően viselkedik.

  • A halmazokban tárolt adatoknak nincs meghatározott sorrendjük
  • Minden adat egyedi, tehát ugyanaz az érték egy halmazban nem fordulhat elő kétszer.
  • A HashSet osztály hatékonyan és gyorsan valósítja meg a halmazműveleteket, mint pl. az unió, metszet, különbség.
  • A halmazban akárhány elemet tárolhatunk. (csak a gép memóriamérete szab korlátot)

Alapvető halmazműveletek

  • h1.Add(x): hozzáad egy x elemet a h1 halmazhoz
  • h1.Remove(x): törli az x elemet a h1 halmazból.
  • h1.Contains(x): lekérdezi, hogy x eleme-e a h1 halmaznak. Ha igen, akkor az eredmény True.
  • h1.Count: megadja a h1 halmazban tárolt elemek számát
  • h1.Clear(): a h1 halmaz összes elemét törli
  • h1.UnionWith(h2): a h1 halmazt egyesíti a h2 halmazzal. Az egyesített halmaz a h1 lesz
  • h1.IntersectWith(h2): a h1 halmazt metszi a h2 halmazzal. Az metszethalmaz a h1 lesz
  • h1.ExceptWith(h2): a h1 halmazt minusz a h2 halmaz. Az különbséghalmaz a h1 lesz

Példaprogram: egész számok halmaza

using System;
using System.Collections.Generic;
class Program
	{
	private static void HalmazMutat(HashSet<int> halmaz)
		{
		Console.Write("{");
		foreach (int elem in halmaz)
			{
			Console.Write(" {0}", elem);
			}
		Console.WriteLine(" }");
		}
	static void Main()
		{
		HashSet<int< A = new HashSet<int>();//egy A nevű, üres számhalmaz
		//hozzáadunk pár elemet:
		A.Add(10);
		A.Add(10);
		A.Add(10);
		A.Add(20);
		A.Add(30);
		A.Add(40);
		//kilistázzuk az A halmazt a konzolra:
		//láthható, hogy noha a 10-et többször is hozzáadtuk, csak 1-szer szerepel az A ban
		HalmazMutat(A); 
		}
	}
