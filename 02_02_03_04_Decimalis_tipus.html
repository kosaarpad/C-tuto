A decimális típus

A decimális típus (decimal) olyan számtípus, amelyet a nagy pontosságú értékek kezelésére hozták létre.
Értéktartománya: (-7.9 x 1028-tól 7.9 x 1028-ig) / (100 tól 1028-ig)
Pontossága 28-29 számjegy.
A decimális értékeket az m vagy M szuffix jelöli. (a szuffix az a betű a számérték végén); 

decimal myMoney = 300.5m;  
//vagy
decimal myMoney = 300.5M;

Mivel értéktípus, ugyanazokat a műveleteket, amiket számértékekkel el lehet végezni, azokat a decimálissal is. példaprogram: 

using System;

class Program
{
	static void Main()
		{
		decimal d = 1.0M;
		// Egyenlőségteszt.
		if (d == 1.0M)
			{
			Console.WriteLine(d);
			}
		decimal y = 0.5M;
		// Értéknövelés:
		y += d;
		Console.WriteLine(y);
		// Értékcsökkentés d kétszeresével:
		y -= (d * 2);
		Console.WriteLine(y);
		}
}

Aritmetika decimálisokkal

using System;

class Program
{
	static void Main()
    {
		Console.WriteLine(decimal.Add(1.1M, 1.3M)); // Add
		Console.WriteLine(decimal.Multiply(5.0M, 2.0M)); // 5 * 2
		Console.WriteLine(decimal.Subtract(2.0M, 1.0M)); // 2 - 1
		Console.WriteLine(decimal.Divide(10M, 2M)); // 10 / 2
    }
}
