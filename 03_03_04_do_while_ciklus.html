A do ... while (hátultesztelős) ciklus

Formátuma:

 
do
     ismételendő_utasítás;
while (logikai_kifejezés);

Magyar olvasata:
Ismételd utasítás-t!
, amíg a logikai_kifejezés értéke igaz marad.

vagy ha több utasítást kell ismételtetni: 

 
do
     { 
      ismételendő_utasítás1;
      ismételendő_utasítás2;
      ismételendő_utasítás3;
	  ...
      ismételendő_utasításN;
     }
while (logikai_kifejezés)

Magyar olvasata:
addig ismételd az utasítás1, utasítás2, utasítás3, ..., utasításN parancssorozatot, amíg a logikai_kifejezés értéke igaz marad!


A fenti példákban a pirossal kiemelt rész a ciklusmag.


Lássunk egy egyszerű példát: 

	
using System;
class Program
{
	static void Main()
	{
	int i;
	i = 0;
	do
		{
		Console.WriteLine(""+i);
		Console.WriteLine("Szia");
		i++;
		}
	while (i < 10)
	}
}
Lássuk tehát, hogyan működik ez a ciklus:
  1. Először a ciklusszámlálót (i-t) inicializáljuk, ami azt jelenti, hogy i megkapja a kezdőértékét, azaz a 0-t
  2. Végrehajtódik a ciklusmag
  3. A ciklusmag végrehajtása során a ciklusváltozó értéke módosul. Itt a módosuláss azt jelenti, hogy i-t növeljük 1-gyel: i++;
  4. Utána kiértékelődik a logikai feltétel: i < 10 . Ha nem igaz, akkor a ciklus véget ér. Ha viszont igaz, akkor ugorj a 2. pontra.
A do ... while ciklusra egy másik példa: visszaszámlálás: 
	
using System;
class Program
{
	static void Main()
	{
	int n;
	Console.WriteLine("Rakétakilövésre felkészülni!");
	n=8;
	do
		{
		Console.Write(""+n+", ";
		n--;
		}
	while (n>0)
	Console.WriteLine("Kilövééés!!!); 
	}
}

Kimenet:

Rakétakilövésre felkészulni!
8, 7, 6, 5, 4, 3, 2, 1, Kilövééés!!!

Az elején felkészülésről szóló üzenet után n a 8 értéket kapja, majd kezdődik a ciklus.
Kiírja az n értékét és a vesszőt, majd n értékét 1-gyel csökkenti, így n értéke tehát 7 lesz.
Ezután kiértékeli a logikai_kifejezes-t 7>0 igaz, így még egyszer végrehajtódik a ciklusmag.
A 8-ik ismétlés után n 0-ra csökken.
Kiértékelődik ismét a logikai kifejezés. 0>0 nem igaz, ezért nem hajtja többé végre a ciklusmagot, hanem
a ciklusmag utáni utasításra ugrik, kiírja az "Kilövééés!!! üzenetet.

A do...while ciklus jellemző tulajdonsága, hogy logikai feltételértékétől függetlenül a ciklusmag egyszer mindenképpen lefut.
Amennyiben a ciklusmag végrehajtása során a feltétel soha nem válik hamissá, akkor a ciklus egy végtelen ciklussá válik.
A programok "lefagyása" általában egy végtelen ciklus következménye. Példa végtelen ciklusra: 

	
using System;
class Program
{
	static void Main()
	{
	int n;
	Console.WriteLine("Rakétakilövésre felkészülni!");
	n=8;
	do
		{
		Console.Write(""+n+", ";
		}
	while (n>0)
	Console.WriteLine("Kilövééés!!!); 
	}
}

Ez majdnem a fenti példa, egyetlen kis eltérés van: az n értéke sohasem változik, mindig 8 marad, és 8 az mindig > 0, ezért a ciklus sohasem ér véget.
A program elkezd 8-asokat írni a konzolra a végtelenségig.
