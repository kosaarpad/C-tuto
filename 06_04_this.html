C# középiskolásoknak

A "this"

A this mutató az aktuális osztály vagy struktúra egy változójára mutat.
Statikus osztály esetében enem használjuk, csak nem statikus osztályoknál.
A this mutatót nem csak a konstruktorban, hanem az osztály tagfüggvényeiben (metódusaiban) is használjuk. Példa: 

using System;
	class Személy 
	{
	string név; 
	int szül_év;
	//Konstruktor
	public Személy(string ujnév, int ujsz_év)
		{
		this.név=ujnév; //<- utal arra, hogy ennek az osztálynak a név változójáról van szó
		this.szül_év=ujsz_év;//<- utal arra, hogy ennek az osztálynak a szül_év változójáról van szó
		}
	public void Bemutatkozik()
		{
		Console.WriteLine("A nevem: "+this.név);//<- utal arra, hogy ennek az osztálynak a név változójáról van szó
		}	
	}
	class Program
	{
  // Main metódus
	static void Main(string[] args)
		{
		Személy sz1 = new Személy("Kiss István", 1998);//
		sz1.Bemutatkozik();
		Console.ReadKey();
		}
}
Itt a Személy osztály NEM statikus, ezért itt a this mutatót kell használnunk az osztályváltozókra.
Itt a Program osztály statikus, ezért itt a this mutatót NEM használjuk az osztályváltozókra.
